# README - /contributions/snapshots/

This folder contains the snapshots in yyyy-mm-dd named folders taken from the editing system (google docs) by one of the editors (Stefan Hagen).
taken at various moments in time in some formats as offered by that platform over time.

The formats (as identified by the suffixes) .docx, .odt, .pdf, and .txt were initially available as a zipped .html format.

Later snapshots also contain a .md (markdown) export as offered by that platform.
